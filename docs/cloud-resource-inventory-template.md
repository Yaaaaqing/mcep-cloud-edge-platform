## 新增资源前必须明确

- 每台虚拟机的实际用途和已安装服务；
- 哪些实例属于数据库虚拟机，哪些资源可以安全复用；
- 是否确需新增 MCEP 应用虚拟机；
- 是否确需新增 MCEP 数据库虚拟机；
- 国产数据库选择达梦或金仓及其版本、许可和兼容性要求；
- 数据备份周期、保留期限、恢复验证与存储方案；
- 资源申请人、责任人、访问权限及安全基线。

完整 IP、账号和资源平台登录信息只能保存在组织批准的私有配置或资产系统中，不得写入公开仓库。
