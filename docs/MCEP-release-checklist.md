## 1. 代码检查

- [ ] `git status` 与 `git diff --stat` 已人工复核。
- [ ] 八个既有路由和 Trace 功能未删除。
- [ ] `npm run lint` 无错误，TypeScript 构建无错误。
- [ ] 未保留未使用组件、重复组件或临时调试代码。

## 2. 敏感信息检查

## 3. 构建检查

- [ ] 已执行 `npm ci`。
